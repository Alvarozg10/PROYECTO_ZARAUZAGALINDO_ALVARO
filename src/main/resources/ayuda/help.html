Sistema de Gestión FCT

Esta aplicación permite gestionar las prácticas en empresa (FCT), incluyendo usuarios, asignaciones y documentación.

Administrador

El administrador puede gestionar los usuarios del sistema:

  • Crear nuevos usuarios
  • Modificar datos existentes
  • Eliminar usuarios

Debe asegurarse de que los datos introducidos son correctos.

Profesor

El profesor puede:

  • Crear nuevas FCT
  • Consultar FCT existentes
  • Gestionar estudiantes

Las fechas de inicio y fin deben ser coherentes.

Estudiante

El estudiante puede:

  • Consultar su empresa asignada
  • Ver datos del tutor
  • Descargar documentos

Debe seleccionar un documento antes de descargarlo.

Tutor Empresa

El tutor puede:

  • Ver alumnos asignados
  • Subir documentación en formato PDF

Solo se permiten archivos PDF.

Errores comunes

  • No rellenar todos los campos
  • No seleccionar elementos en tablas
  • Subir archivos no válidos

El sistema mostrará mensajes de error en estos casos.
